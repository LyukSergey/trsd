CORS Demo

Ця сторінка: http://localhost:3000 (інший origin)
API сервер: http://localhost:8080
Origin різні → браузер застосовує Same-Origin Policy → потрібен CORS
Натисніть кнопку щоб зробити запит...

Що відбувається:

  1. Ця сторінка працює на http://localhost:3000
  2. JavaScript робить fetch() до http://localhost:8080
  3. Браузер бачить що origin різні → перевіряє CORS headers у відповіді
  4. Без CORS конфігурації на сервері → браузер блокує відповідь
  5. З CORS конфігурацією → сервер додає Access-Control-Allow-Origin → браузер дозволяє

Зверніть увагу:

  • Відкрийте F12 → Network → подивіться на запит
  • Без CORS: запит відправляється на сервер, але браузер блокує відповідь для JavaScript
  • Для POST з Content-Type: application/json браузер спочатку шле OPTIONS (preflight request)

Як увімкнути CORS на сервері:

// SecurityConfig.java
.cors(cors -> cors.configurationSource(corsConfig()))

// або на контролері:
@CrossOrigin(origins = "http://localhost:3000")
